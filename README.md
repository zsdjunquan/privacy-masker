# Privacy Masker

Windows 桌面隐私遮罩 V0.1 原型。

## 当前能力

- 一键开启或关闭隐私模式
- 全局快捷键 `Ctrl + Alt + P`
- 系统托盘常驻
- 自动扫描可见窗口
- 识别微信、企业微信、QQ、TIM 窗口
- 用置顶半透明遮罩覆盖目标窗口，并跟随窗口移动和缩放
- 可选择是否在本机显示遮罩
- 支持选择本地 PNG/JPG/BMP 作为自定义遮罩素材
- 支持内置遮罩背景和自定义文案

## 自定义遮罩素材

在主界面点击“选择图片”可以选择本地素材。素材会同时作用于：

- 本机遮罩层
- 安全共享窗口里的遮罩区域

当前原型使用“完整适应”方式显示素材，避免把图片中的文字或主体裁掉。

如果不使用自定义图片，可以选择内置遮罩背景，并设置类似“不可以偷看哦”的提示文案。

## 运行方式

需要 Windows 和 .NET 8 SDK。

```powershell
dotnet run
```

如果旧终端还没有刷新 PATH，也可以直接运行：

```powershell
.\run.cmd
```

## V0.1 限制

这个原型优先保护“共享整个桌面”的场景。普通桌面共享时，对方看到的画面通常与本机屏幕一致，所以本机显示遮罩时，对方也会看到遮罩。

如果关闭“本机显示遮罩”，本机不会被挡住，但普通桌面共享里对方也不会看到遮罩。真正的“本机无遮罩、对方有遮罩”需要改成安全共享窗口、虚拟屏幕、会议软件插件或底层采集链路处理。

如果会议软件只共享某一个单独窗口，会议软件可能直接采集原窗口内容，独立遮罩层不一定会被采集进去。

下一步适合补充：

- 自定义应用规则
- 自定义固定遮罩区域
- 自动检测会议软件
- 自动最小化敏感窗口
- 多屏幕与 DPI 细节优化
